# Primera Entrega - Cálculo de Límites

Este repositorio contiene un programa en C# para **calcular límites aproximados de funciones**.  
El código fue desarrollado con Visual Studio 2022 y utilizara la librería `Math` para los cálculos proximamente.  

Incluye:
- Archivo `Program.cs` con la lógica del programa.
- Documentación explicando cómo funciona el código y cómo calcular límites.

Creado como entrega de la materia **Cálculo Infinitesimal**.
